# Interactive Task Dashboard

A modern and responsive Task Management Dashboard built using React.  
This project demonstrates strong understanding of state management, dynamic rendering, reusable components, and clean UI architecture.

---

## 🚀 Features

- Add tasks dynamically
- Delete tasks
- Toggle task completion
- Dark / Light theme toggle
- Task summary (Total & Completed count)
- Persistent data using localStorage
- Clean desktop-focused dashboard layout
- Responsive design improvements

## ⚙️ Project Setup

1. Clone the repository:
git clone https://github.com/ananya-kushi07/Interactive-Dashboard.git

2. Navigate to project folder:
cd interactive-task-dashboard

3. Install dependencies:
npm install

4. Run the project:
npm start

The application will run at:
http://localhost:3000

---

## 📊 How It Works

- Tasks are stored in React state.
- When a task is added, state updates dynamically.
- useEffect stores tasks in localStorage.
- On reload, saved tasks are retrieved automatically.
- Theme toggle switches CSS classes for dark/light mode.
- Task summary updates automatically based on state.
